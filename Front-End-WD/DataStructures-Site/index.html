Data Structure Guide

Welcome to my data structure guide

The following is a list of common data structures along with explanations

Arrays

            +----+----+----+----+----+
            | 10 | 20 | 30 | 40 | 50 |
            +----+----+----+----+----+
               0    1    2    3    4

Overview

This is an array data structure.

It is a an ordered collection of what are called elements

An element is any value of a certain type, such as number, boolean, even another array

In the example above the elements are the numbers in the boxes

Each box represents a place in the array that contains a stored element

At the bottom of the image you'll see it's labeled with a number

These numbers are called indices, the index represents the position of the element in the array

As mentioned, Arrays are ordered starting from index 0, and elements are next to each other

Code Examples Javascript

let numberList = [10, 20, 30, 40, 50]

This is how you would make the above array

let firstNumber = numberList[0]

This is called index notation you can access elements in the array using it's index

Linked List

                +----+    +----+    +----+    +----+    +----+
                        | 10 | -> | 20 | -> | 30 | -> | 40 | -> | 50 | -> NULL
                +----+    +----+    +----+    +----+    +----+

Overview

This is a Linked List data structure.

Linked List consist of Nodes, each node has a velue, which is the number in the box

Unlike arrays they are not stored next to each other in memory but has a pointer which

is represented by the arrows between the boxes, the pointer is the address of the next value in the list
